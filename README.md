# PapaLinuxTelepito
Simple packed app installer(curl) for my Grandpa
Eng: My grandpas laptop sucks on windows, so he installed Lubuntu, but he doesnt like it. I recommended arch and that I will write him a script that he can copy and paste it into terminal and use it simply.
Hun: A papám laptopja lassú windows-on, ezért feltette a Lubuntut, de nem nagyon szereti. Én ajánlottam neki az Arch-ot, és írok neki egy script-et, hogy csak bemásolja és feltelepítse neki a programokat.
This is the most simple program ever so my dumbass doesnt have to maintain it much
Ez a legegyszerűbb script, mert lusta vagyok frissíteni


Curl telepítő/installer: curl -fsSL https://github.com/papatelepito.sh
